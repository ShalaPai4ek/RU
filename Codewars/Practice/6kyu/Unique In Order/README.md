# Unique In Order

## Условие задачи
Написать функцию, которая будет преобразовывать повторения 1-го и тогоже символа в строке друг за другом в 1 символ. Результат должен выводить список символов по отдельности.

## Решение
```python
def unique_in_order(sequence):
    uniq = []
    for i in range(len(sequence)):
        if i == 0 or sequence[i] != sequence[i-1]:
            uniq.append(sequence[i])
    return uniq
```

## Объяснение
Задаём переменную uniq в которую будем записывать результат в виде списка. Далее задаём цикл for который пройдёт по всем символам строки. В цикле задано условие: если символ самый первый или сам символ не равен предыдущему, то добавь его в список uniq. В конце возвращаем список. 

## Нюансы
Здесь не нужно использовать .lower(), так как по условию регистр должен учитываться.
