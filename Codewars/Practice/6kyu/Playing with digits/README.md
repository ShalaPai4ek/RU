## Условие задачи
Написать функцию, которая принимает 2 аргумента в виде чисел. 1 аргумент исходное число, а 2 аргумент это степень. Числа исходного числа, должны возводиться в степень, которая будет на 1 больше с каждым следующим элементом. После этого они должны складываться. Сумма должна делиться на исходное число, если же нет, то нужно вернуть '-1'.

## Решение
```python
def dig_pow(n, p):
    numb = []
    owo = 0
    for i in str(n):
        numb.append(int(i))
    for i in numb:
        owo += i**p
        p += 1
    if owo % n == 0:
        return owo//n
    else:
        return -1
```

## Объяснение
Задаём список чтобы разбить исходное число на элементы для удобства. Также задаём переменную, в которую будет записываться сумма. Далее с помощью цикла for добавляем элементы в список. Следующий цикл for проходит по элементам списка, возводит их в степень и прибавляет к переменной 'owo'. В конце цикла обязательно переменная увеличивается на 1. Последним этапом идёт проверка. Если полученное число делится на исходное без остатка, то верни результат деления. В остальных случаях верни -1.

## Нюансы
