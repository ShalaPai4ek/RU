# Count the smiley faces!

## Условие задачи
Нужно посчитать количество правильных смайликов в списке. Правильными считаются только те смайлики, что соблюдают все условия: 1) Сначала идут глаза ':' или ';'. 2) Далее опционально идёт нос '-' или '~'. 3) В самом конце обязательно должен идти рот ')' или 'D'. Порядок должен соблюдаться, а символы не прописанные в условии считаются ошибкой. 

## Решение
```python
def count_smileys(arr):
    count = 0
    eyes = {':', ';'}
    noses = {'-', '~'}
    mouths = {')', 'D'}
  
    for i in arr:
        if len(i) == 2:
             if i[0] in eyes and i[1] in mouths:
                count += 1
        elif len(i) == 3:
            if i[0] in eyes and i[1] in noses and i[2] in mouths:
                count += 1
                
    return count
```

## Объяснение
Сначала задаём переменную в которой будем считать количество правильных смайликов. Далее прописываем неупорядоченные коллекции того, что является глазами, носом и ртом. С помощью цикла for проходим по заданному списку. В цикле проверяем длину смайлика, если 2, то используем шаблон для 2-х символов. Тоесть если первый элемент смайлика это глаза, а следующий это рот, то прибавляем к переменной 'count' +1. В ином случае если символов 3 используем шаблон для 3-х символов. Тоесть если первый элемент смайлика это глаза, второй нос, а третий рот, то прибавляем к переменной 'count' +1. После завершения цикла возвращаем 'count'.

## Нюансы
Можно было не использовать множество проверок, а обойтись 'raw' строкой. Которая бы указывала, что первый элемент должен быть глазами, второй опционально это нос, а третий обязательно рот.
