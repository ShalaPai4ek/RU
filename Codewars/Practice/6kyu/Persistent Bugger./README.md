# Persistent Bugger.

## Условие задачи
Написать функцию, которая будет считать сколько раз нужно умножить цифры числа, пока результат не будет равен одной цифре.

## Решение
```python
def persistence(n):
    a = 0
    while n > 9:
        b = 1
        for i in str(n):
            b *= int(i)
        a += 1
    return a
```

## Объяснение
Задаём переменную (a), в которую будем записывать результат посчёта. Далее задаём цикл, который будет работать пока заданное число больше 9, тоесть пока число больше 1 символа. Задаём переменную (b) в которой будет производиться умножение. Дальше идёт цикл для перебора цифр в числе. В самом конце производим подсчёт и возвращаем результат.
