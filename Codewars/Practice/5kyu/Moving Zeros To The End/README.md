# Moving Zeros To The End

## Условие задачи
Создать функцию, которая будет перемещать все нули в конец списка.

## Решение
```python
def move_zeros(lst):
    non_zeros = [x for x in lst if x != 0]
    zeros = [x for x in lst if x == 0]
    return non_zeros + zeros
```

## Объяснение
Сначала мы задаём переменную 'non_zeros' в которую мы записываем список сгенерированный циклом. Где 'x' — то что мы записываем, 'lst' — объект который мы перебираем, 'if x != 0' — условие добавления. Таким образом мы создаём список из чисел, которые на равны нулю. Следующая переменная 'zeros' будет хранить в себе список нулей, создаётся он по тому же принципу. В конце мы возвращаем сумму этих списков, где нули оказываются в конце.  

## Нюансы
Этот же код, можно реализовать циклами с методом '.append()', но так получилось бы длиннее.
