# The Hashtag Generator

## Условие задачи
Написать функцию, которая будет добавлять в начало строки '#' и убирать из результата ' '. Также нужно, чтобы все символы были в ниждем регистре. Если результат будет больше 140 символов или в результате выйдет только '#', нужно вывести 'False'. 

## Решение
```python
def generate_hashtag(s):
    s = '#' + s.title().replace(' ', '')
    if len(s) > 140 or len(s) == 1:
        return False
    else:
        return s
```

## Объяснение
Перезаписываем переменную, добавляя в начало '#'. Складываем '#' с исходным значением из переменной, используя метод .title() и .replace(). В свою же очередь .title() отвечает за преобразование в нижний регистр, а в .replace() мы указываем ('на что заменить', 'что мы заменяем'). Так мы избавляемся от пробелов.  
Далее просто идёт проверка на количество символов с помощью len(). Если полученная длина строки больше 140 или равна 1 символу, то верни 'False'. Во всех других случаях возвращаем результат.
## Нюансы
Проверку можно было указать в одну строку в самом return.
