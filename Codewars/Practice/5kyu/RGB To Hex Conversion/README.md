# RGB To Hex Conversion

## Условие задачи
Сделать фунцию, которая принимает три числовых аргумента, и переводит их в HEX код для цвета. Вывод должен быть в формате строки.

## Решение
```python
def rgb(r, g, b):
    rgb = [max(0, min(i, 255)) for i in (r, g, b)]
    return "{:02X}{:02X}{:02X}".format(*rgb)
```

## Объяснение
Задаём переменную в которую записываем сгенерированный список. Мы указываем что число обязательно должно быть от 0 до 255. Если это не так, то число округлится до 0 или 255 в зависимости от того, что ближе. В конце возвращаем отформатированную строку. {:02X} означает, что число должно быть ровно 2 символа, если это не так, то оно заполняется 0 слева. X - указывает на перевод в HEX с использованием заглавных букв. Метод .format() распаковывает список в строку
